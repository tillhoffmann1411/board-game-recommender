# Social Media Images & Icons

This document outlines the images needed for proper social media sharing and favicon support.

## Required Images

### 1. Open Graph Image (`/public/og-image.png`)
- **Size**: 1200x630 pixels
- **Format**: PNG
- **Purpose**: Displayed when sharing links on Facebook, LinkedIn, Twitter, etc.
- **Design**: Should include:
  - Brand colors: Purple (#9333EA) to Pink (#EC4899) gradient
  - Dice/board game iconography
  - "Board Game Recommender" title
  - Tagline: "Discover Your Next Favorite Game"
  - Clean, modern design matching the website aesthetic

### 2. Favicon (`/public/favicon.ico`)
- **Size**: 16x16, 32x32, 48x48 (multi-size ICO file)
- **Format**: ICO
- **Purpose**: Browser tab icon

### 3. SVG Icon (`/public/icon.svg`)
- **Size**: Scalable vector
- **Format**: SVG
- **Purpose**: Modern browser icon support
- **Design**: Simple dice icon with purple-pink gradient background

### 4. Apple Touch Icon (`/public/apple-icon.png`)
- **Size**: 180x180 pixels
- **Format**: PNG
- **Purpose**: iOS home screen icon

### 5. PWA Icons
- **192x192** (`/public/icon-192.png`): For Android home screen
- **512x512** (`/public/icon-512.png`): For Android splash screen

## Design Guidelines

- Use the brand gradient: Purple (#9333EA) to Pink (#EC4899)
- Keep designs simple and recognizable at small sizes
- Ensure high contrast for visibility
- Match the playful, modern aesthetic of the website
- Consider the dice icon from the header as a design element
